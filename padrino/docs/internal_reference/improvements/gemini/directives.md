# Gemini CLI Directives

These instructions are foundational for the Gemini CLI when working on the **Don** project.

## Mandatory Skills
- **Go Standards:** Always activate the `go-standards` skill before performing any Go-related coding tasks.
- **Semantic Commits:** Always use the `semantic-commit` skill for all commits to maintain a consistent history.

## Workflow Expectations
- Maintain the Hexagonal Architecture integrity.
- Ensure all new features include corresponding tests.
- Verify changes with the project's linting and testing commands.
